# Práctico Integrador — Git y GitHub

## Descripción
Repositorio de práctica para las clases 1, 2 y 3 de Software Factory.
Contiene ejercicios de Git, GitHub y Python básico.

## Contenido
- `mi_info.py` — Script con información personal y hobbies
- `quiz_git.py` — Quiz interactivo sobre conceptos de Git
- `.gitignore` — Archivos ignorados por Git

## Cómo ejecutar

python mi_info.py
python quiz_git.py

## Tecnologías
- Python 3
- Git y GitHub
